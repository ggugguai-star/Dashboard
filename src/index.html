위젯 설정

Google 캘린더처럼 여러 개를 선택할 수 있어요. 위젯마다 다르게 설정 가능합니다.

Gemini 무료 API 키 발급

  1. 개인 Gmail 계정으로 Google AI Studio에 접속합니다.
  2. 좌측 메뉴에서 API 키를 선택합니다.
  3. 키 만들기 → 프로젝트 선택 후 키를 복사합니다.
  4. 위젯 ⚙ 설정에 키를 붙여넣고 저장합니다.
교육청 관리 계정은 AI Studio가 차단된 경우가 많습니다. Gemini 키는 대시보드 Google 로그인과 별개이며, 개인 Gmail으로 발급하는 것을 권장합니다.
